# aram-starter

Aram Labs design system conventions, subdomain patterns, and auth-gate usage for Claude Code.

## What's Included

- **Skill: aram-starter** — Teaches Claude about aramlabs.css design tokens, the `*.aramlabs.ca` subdomain system, auth-gate integration, and project organization conventions.

## Usage

After installing this plugin, Claude Code will automatically apply Aram Labs conventions when building sites or tools for Aram Labs clients.

The skill activates when working on projects for: WealthBlume, L Squared, Arbour Oak, Grizzly, TBDC, or any Aram Labs internal tool.
